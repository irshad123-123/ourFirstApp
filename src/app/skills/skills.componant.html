Some Common Full Form

Sr.No Short Form Full Form Use
1 JS JavaScript Programming language for web development
2 ECMA European Computer Manufacturers Association The organization that standardizes JavaScript (ECMAScript)
3 ES ECMAScript The official standard/specification name of JavaScript
4 DOM Document Object Model Structure representing HTML elements as objects
5 BOM Browser Object Model Represents browser-related objects like window, navigator, etc.
6 JSON JavaScript Object Notation Lightweight format to store and exchange data
7 AJAX Asynchronous JavaScript and XML Technique for making async web requests (now usually with JSON)
8 HTTP HyperText Transfer Protocol Protocol for communication between browser and server
9 URL Uniform Resource Locator Address of a resource on the internet
10 CRUD Create, Read, Update, Delete Basic database operations
11 IIFE Immediately Invoked Function Expression Function that runs immediately after being defined
12 OOP Object-Oriented Programming Programming style using objects and classes
